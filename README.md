# EVO DNA Atlas

This repository is the public release surface for the EVO DNA Atlas:

- the static website;
- the compact, publication-approved gene catalog;
- the public provenance manifest and layered validation passports.

The training engine, model implementation, DNA search and evolution code,
exporters, tests, raw runs, and internal evidence bundles are maintained in a
separate private repository and are not distributed here.

Visit the atlas at <https://terrenceyeyang.github.io/EVO/>.
